# UWE_Systems_development_GVCC_Case_
### pipenv 
#### linux
- run `pipenv sync` to keep your venv the same as everyone else
- `pipenv run python main.py` runs main.py in the venv
####  windows
- run `python -m pipenv sync`to keep your venv the same as everyone else
- `python -m pipenv run python src/main.py` runs main.py in the venv
